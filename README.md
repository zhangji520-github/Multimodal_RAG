# Multimodal RAG

这是一个多模态检索增强生成 (Multimodal Retrieval-Augmented Generation) 项目。

## 项目简介

本项目旨在构建一个支持文本、图像等多种模态数据的检索增强生成系统。

## 功能特性

- 多模态数据处理
- 检索增强生成
- 基于 LangGraph 的工作流

## 安装依赖

```bash
pip install -r requirements.txt
```
